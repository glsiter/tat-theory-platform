特质激发理论 (TAT) 核心概念

🧠理论定义

特质激发理论（TAT）是由Tett和Guterman（2000）提出的重要人格心理学理论， 旨在解释人格特质如何在特定情境中被激发并影响个体行为。

🎯核心观点

人格特质只有在相关的情境线索存在时才会被激发，进而预测和影响个体的行为表现。 这一观点整合了特质论和情境论的优势。

核心研究发现

支持的假设

H1: 外向性 → 有形性 (支持)

路径系数: β = -0.215, p < 0.01

解释: 内向客户更重视酒店的物理设施和环境

理论依据: 内向者对外部刺激更敏感，更关注有形环境

H5: 外向性 → 同理心 (支持)

路径系数: β = 0.109, p < 0.05

解释: 外向客户更重视员工的个性化服务和关注

理论依据: 外向者需要更多社交互动和个人关注

不支持的假设

H2: 外向性 → 可靠性

β = -0.018, p = 0.743 (不显著)

H3: 外向性 → 响应性

β = 0.086, p = 0.076 (边缘显著)

H4: 外向性 → 保证性

β = 0.099, p = 0.052 (边缘显著)

TAT理论概念词典

特质 (Trait)

相对稳定的个体差异变量，代表个体的行为倾向和心理特征。

  • • 跨时间和情境的稳定性
  • • 个体间的差异性
  • • 对行为的预测作用

激发 (Activation)

特质在特定情境下被唤醒并影响行为的动态过程。

  • • 情境依赖性
  • • 选择性激发
  • • 强度变化

情境线索 (Situational Cues)

环境中能够激发特定特质的刺激因素和环境特征。

  • • 任务特征
  • • 社会环境
  • • 物理环境
